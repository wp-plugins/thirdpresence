Allows the user to embed Thirdpresence video by entering a shortcode ([thirdpresence]) into the post area.

== Description ==

1. Enter the [thirdpresence video="videoid" /] into any post. The parameter "videoid" is the id number of the video.
1. When the post is accessed, a thumbnail image of the video will be displayed inline with the rest of the article.
1. By clicking on the thumbnail, the video will play on the mobile device.

More detailed instructions can be found from [Thirdpresence Wiki](http://wiki.thirdpresence.com/index.php/WordPress_Plugin_Description)

== Installation ==

1. Unzip `thirdpresence-plugin.zip` and upload the contained files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the plugin through the 'Plugins' menu in WordPress to set your Thirdpresence account id
